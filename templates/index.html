Policy Cluster Reference Guide

The K-Means model groups households into 4 segments based on income, cost burden, age, and housing type. Each cluster maps to a targeted policy intervention.

⚙️ Model Details

Algorithm K-Means (k=4)
Data Source U.S. American Housing Survey (AHS)
Features 15 (income, burden, age, bedrooms, region, AMI…)
Scaling MinMax Normalization
Cluster Selection Elbow Method + Silhouette Score
Burden Threshold >30% = Burdened, >50% = Severely Burdened
